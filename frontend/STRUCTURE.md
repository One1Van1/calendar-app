# Frontend Structure

## Архитектура

Компонентная архитектура с разделением логики и отображения:

```
frontend/
├── renderer/
│   ├── components/           # Компоненты приложения
│   │   ├── Calendar/         # Компонент календаря
│   │   │   ├── calendar.logic.js    # Логика календаря
│   │   │   └── calendar.view.js     # Отображение календаря
│   │   ├── Header/           # Компонент шапки
│   │   │   ├── header.logic.js      # Логика навигации и темы
│   │   │   └── header.view.js       # Отображение шапки
│   │   ├── Sidebar/          # Компонент боковой панели
│   │   │   ├── sidebar.logic.js     # Логика категорий
│   │   │   └── sidebar.view.js      # Отображение категорий
│   │   └── Modal/            # Компонент модального окна
│   │       ├── modal.logic.js       # Логика создания событий
│   │       └── modal.view.js        # Отображение формы
│   ├── services/             # Сервисы для API
│   │   └── api.service.js    # Работа с REST API
│   ├── utils/                # Утилиты
│   │   ├── date.utils.js     # Работа с датами
│   │   └── theme.utils.js    # Работа с темами
│   ├── app.new.js            # Главный файл приложения
│   ├── index.html            # HTML страница
│   └── styles.css            # Стили
```

## Принципы разделения

### Logic (Логика)
- Управление состоянием компонента
- Обработка событий
- Бизнес-логика
- Взаимодействие с API через сервисы

### View (Отображение)
- Рендеринг HTML
- Манипуляция DOM
- Визуальные обновления
- Чистые функции без состояния

## Компоненты

### HeaderComponent
- Навигация по месяцам
- Переключение темы
- Кнопка "Сегодня"

### CalendarComponent
- Отображение сетки календаря
- Загрузка событий
- Рендеринг дней с событиями

### SidebarComponent
- Список категорий
- Мини-календарь
- Фильтрация

### ModalComponent
- Создание событий
- Валидация формы
- Работа с категориями

## Сервисы

### apiService
- Все HTTP запросы к backend
- CRUD операции для Events, Categories, Reminders
- Обработка ошибок

## Утилиты

### dateUtils
- Форматирование дат
- Сравнение дат
- Работа с месяцами

### themeUtils
- Переключение темы
- Сохранение в localStorage
- Применение темы
